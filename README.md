
# ⚡ GigCredit
### Micro-loan eligibility tool for India's 15 million gig workers

🌐 **Live App:** https://gigcredit-oi9q.vercel.app

---

## The Problem
Swiggy drivers, Ola partners, Urban Company professionals earn
real documented income — but banks auto-reject them because
they have no salary slips. No rejection reason is ever given.

## What GigCredit Does
- Scores your loan eligibility using 6 alternative data factors
- Explains exactly why banks reject you — in plain language
- Gives a week-by-week 90-day action plan with rupee-specific steps
- Matches you to real loans you qualify for right now
- What-if simulator to see score changes in real time
- Works in English, Hindi and Tamil

## Built With
- React
- Claude AI API (claude-sonnet-4-20250514)
- Vercel (deployment)
